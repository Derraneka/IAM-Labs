# Lab 01: Okta Org Customization

## Objective
Customize an Okta organization branding and account settings to gain hands-on experience with Okta administration.

---

## Skills Practiced
- Okta Admin Console navigation
- Branding customization
- Theme configuration
- Email template customization
- Domain settings review
- Account settings management
- End-user dashboard verification

---

## Lab Scenario
In this lab, I customized an Okta organization by modifying branding elements, reviewing email templates, exploring domain settings, and verifying changes from the end-user perspective.

---

## Tasks Completed

### 1. Customized Organization Branding
Configured branding settings in the Okta Admin Console:
- Updated organization logo
- Modified primary color
- Modified secondary color
- Updated favicon
- Reviewed branding preview

---

### 2. Reviewed Email Customization
Explored email customization options:
- Opened email templates
- Reviewed available branding options
- Examined HTML customization capability

---

### 3. Reviewed Domain Settings
Navigated to:
Settings → Customizations → Brands → Domains
Actions:
- Reviewed domain configuration options
- Explored customization limitations for default Okta subdomain

---

### 4. Modified Account Settings
Navigated to:
Settings → Account

Reviewed:
- Admin email notifications
- Organization account settings
- General administrative options

---

### 5. Verified End-User Experience
Opened the End-User Dashboard and confirmed:
- Branding changes displayed correctly
- Updated logo visible
- Color changes reflected
- End-user interface customization applied

---

- Identity branding and user experience

- End-user access portal management
- Identity platform governance basics

Successfully completed a hands-on Okta administration lab focused on organization customization and end-user branding verification.
